Konfiguracja synchronizacji

Aby dane synchronizowały się między telefonem a komputerem, podaj swój GitHub Personal Access Token i Gist ID. Dane są zapisywane tylko w Twojej przeglądarce i wysyłane wyłącznie do Twojego prywatnego Gist.

Jak uzyskać Token i Gist ID?
Szczegółowa instrukcja jest w pliku INSTRUKCJA.md dołączonym do tego projektu.
