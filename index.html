☁ Synchronizacja między urządzeniami

Podaj swój GitHub Token i Gist ID — dane będą automatycznie synchronizowane między telefonem a komputerem. Token jest przechowywany tylko w Twojej przeglądarce.

1. Idź na gist.github.com → utwórz nowy secret gist z plikiem moje-cele.json
2. Skopiuj ID z adresu URL (ciąg po Twoim loginie)
3. Idź na github.com/settings/tokens → nowy token → zaznacz tylko gist
4. Wklej oba poniżej i kliknij Zapisz
